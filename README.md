### 4. 중요도 시스템 

#### 중요도 계산 (0-100점)
```
기본 점수 (심각도):
  🔴🔴🔴 CRITICAL: 100점
  🔴🔴 HIGH: 80점
  🟡 MEDIUM: 50점
  🟢 LOW: 20점

추가 점수:
  + 정확도 (0-20점): 매치 텍스트 길이 / 전체 라인 길이
  + 매치 개수 (0-10점): 같은 오류가 여러 번 나타나면 가산

최종 중요도 = 기본 점수 + 정확도 점수 + 매치 개수 점수 (최대 100점)
```

#### 중요도 표시
```
[██████████] 100% - 매우 중요 (즉시 해결)
[████████░░] 80% - 중요 (빠른 해결)
[██████░░░░] 60% - 보통 (해결 권장)
[████░░░░░░] 40% - 낮음 (선택적 해결)
```

### 5. 심각도 분류 (4단계)

#### 🔴🔴🔴 CRITICAL (매우 심각)
- 즉시 해결 필요
- 게임을 할 수 없음
- 예: 메모리 부족, 반복 크래시

#### 🔴🔴 HIGH (심각)
- 빠른 해결 필요
- 게임 플레이 불가능
- 예: 모드 충돌, 그래픽 오류, 자바 오류

#### 🟡 MEDIUM (중간)
- 해결 권장
- 게임 플레이 가능하지만 문제 있음
- 예: 네트워크 오류, 버전 불일치, 청크 로딩 오류

#### 🟢 LOW (낮음)
- 선택적 해결
- 게임 플레이에 큰 영향 없음
- 예: 텍스처 오류, 사운드 오류, 스킨 오류

### 3. 더 상세한 설명

```
메모리 부족
마인크래프트에 할당된 메모리가 부족합니다. 게임이 더 이상 메모리를 할당받지 못해 튕깁니다.

심각도: 🔴🔴🔴 매우 심각 - 즉시 해결 필요

발견 위치: 줄 4: java.lang.OutOfMemoryError: Java heap space

해결 방법 (순서대로 시도하세요):
1. 런처에서 할당 메모리를 늘려보세요 (최소 2GB, 권장 4GB)
   - 런처 > 설정 > Java 설정 > JVM 인수
   - -Xmx2G를 -Xmx4G로 변경 (4GB로 설정)
2. 불필요한 프로그램을 종료하세요 (크롬, 디스코드 등)
...
```

## 📊 오류 패턴 개선

### 더 정확한 정규식 패턴

#### 메모리 부족
```
개선 전: (OutOfMemoryError|java\.lang\.OutOfMemoryError)
개선 후: (OutOfMemoryError|java\.lang\.OutOfMemoryError|heap space|GC overhead)
```

#### 그래픽 오류
```
개선 전: (OpenGL|graphics|render|shader|display|video)
개선 후: (OpenGL error|GLFW error|graphics error|render error|shader error|display error|GLX error)
```

#### 네트워크 오류
```
개선 전: (Connection|socket|network|timeout|refused)
개선 후: (Connection refused|Connection timeout|socket error|network error|unreachable|Connection reset)
```
